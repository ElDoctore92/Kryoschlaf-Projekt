# Kryoschlaf Project - Documentation

**Kurzbeschreibung**  
Digital Twin Simulation für Langzeit‑Kryoschlaf mit Fokus auf Muskelabbau, Perfusionssteuerung, EMS‑Interventionen und KI‑gestützte Aufwachprotokolle. Dieses öffentliche Repository enthält Dokumentation, ein Whitepaper, Beispielkonfigurationen und Beispieloutputs. Der vollständige Simulationscode und Rohdaten sind privat gehalten.
